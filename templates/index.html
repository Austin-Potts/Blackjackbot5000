Blackjack Machine Learning

  • Team

This project utilizes the reinforcement learning methodology of machine learning to train an algorithm by playing a simulation blackjack game 1 million times. The algorithm learned to implement strategies by being rewarded each time it won a hand. The visuals in the slideshow below highlight our key findings, and the app below will make strategic suggestions based on the algorithm's opitimized strategy.
